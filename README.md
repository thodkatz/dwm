# dwm - dynamic window manager

dwm is an extremely fast, small, and dynamic window manager for X.

List of [patches](https://dwm.suckless.org/patches/):

- bottomstack
- alternativetags
- gridmode

- libxft-bgra
- ttf-joypixels (font for emoji)
